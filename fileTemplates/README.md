## Updating
 
- Go to your project directory:
 
     ```bash
     composer update 
     ```
 
- Pull the latest code from repository by Git (If you want latest `master` branch):
 
     ```bash
     git checkout master
     git pull -r
     ```
 
     Or pull latest version:
 
     ```bash
     git fetch
     git checkout <version>
     ```
 
- Update the Composer dependencies: `composer install`
